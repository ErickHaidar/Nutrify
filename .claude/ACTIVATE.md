Sebelum mulai mengerjakan apapun, lakukan langkah berikut secara otomatis:

1. Baca file `CLAUDE.md` di root project untuk memahami konteks dan aturan orchestration skill
2. Baca semua file `SKILL.md` di dalam folder `.claude/skills/` (ada 18 skills)
3. Setelah membaca, berikan briefing singkat:
   - Nama project dan tech stack
   - 3-5 skills paling relevan untuk sesi ini
   - Tanyakan: "Apa yang ingin kita kerjakan hari ini?"

Selama sesi ini, selalu aktifkan skill yang relevan secara otomatis berdasarkan konteks task — jangan tunggu diminta.
